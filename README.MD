# llm2rtf

Конвертер математического Markdown нейросетей (Gemini/Google AI Studio и др.) в формат RTF, который корректно отображается в Word, WordPad и других текстовых редакторах. C **сохранением формул** (в упрощенном виде), **таблиц** и **форматирования**.
### Для чего нужен?
Нейросети выдают математические ответы в формате Markdown и LaTeX. При обычном копировании в редакторы типа Word, особенно старых версий, формулы превращаются в код (LaTex), а таблицы и заголовки теряют структуру (md).

Скрипт берет текст из буфера обмена, вычищает мусор LaTeX, конвертирует сложные формулы в символы Unicode и упаковывает всё в Rich Text Format (RTF), который поддерживается практически всеми редакторами текста.

*Вы получаете не программный код, а человекочитаемые формулы с использованием греческих букв, степеней и математических знаков.*

**Исходный текст от LLM:**
> В $\triangle ABC$ известно: $\angle \alpha = 60^\circ$, сторона $AB = \mathbf{10\sqrt{3}}\text{ см}$. Найти высоту $h = \frac{AB \cdot \sin(\alpha)}{\sqrt{x^2 + y^2}}$.

Результат при копировании из LLM в редактор **без конвертации**:
> `В $\triangle ABC$ известно: $\angle \alpha = 60^\circ$, сторона $AB = \mathbf{10\sqrt{3}}\text{ см}$. Найти высоту $h = \frac{AB \cdot \sin(\alpha)}{\sqrt{x^2 + y^2}}$.`

Результат **после конвертации**:
> В *△ ABC* известно: *∠ α = 60°*, сторона *AB = **10√3** см*. Найти высоту *h = (AB ⋅ sin(α)) / √(x² + y²)*.

### Какие редакторы поддерживаются
RTF — это стандарт форматирования, уходящий корнями в мохнатый 1987 год. Результат работы скрипта можно вставлять в:
- Microsoft Word (любых версий, включая 97, 2003, 2007 и современные).
- WordPad
- LibreOffice / OpenOffice.
- Старые редакторы типа AbiWord.
- Текстовые процессоры на macOS (TextEdit) и Linux.

### Как это работает
1. Вы копируете текст из окна чата с нейросетью (`Copy as markdown`).
2. Запускаете скрипт `llm2rtf.py` (двойным кликом, иконкой на панели, горячей клавишей или через терминал: `python llm2rtf.py` - тут как придумаете).
3. Скрипт обрабатывает текст и обновляет буфер обмена.
4. Вы вставляете результат в свой редактор (`Ctrl+V`).

### Что умеет конвертер
- Математические символы: заменяет команды типа \alpha, \triangle, \perp на реальные знаки α, △, ⊥.
- Степени и индексы: превращает `x^2` в `x²` и `x_{12}` в `x¹²`.
- Дроби и корни: преобразует сложные дробные конструкции в читаемый линейный вид с использованием скобок и знака √.
- Таблицы: преобразует Markdown-таблицы в настоящие сетки Word с границами.
- Форматирование: сохраняет жирность, курсив (математические переменные автоматически становятся курсивом) и три уровня заголовков.

### Ограничения
Инструмент ориентирован на быструю подготовку текста и использует "линейную" запись формул. Сложная многоэтажная верстка упрощается для сохранения совместимости со старыми редакторами. Например: интегралы, суммы с индексами сверху/снизу и матрицы не отрисовываются графически.

**Примеры упрощений:**
1. **Дроби**: Сложные дроби записываются через слеш.
   - Было: `\frac{a + b}{c + d}`
   - Стало: `(a + b) / (c + d)`
2. **Корни**:
   - Было: `\sqrt{x^2 + y^2}`
   - Стало: `√(x² + y²)`
