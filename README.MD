Нейросети выдают математические ответы в формате Markdown и LaTeX. При обычном копировании в редакторы типа Word, особенно старых версий, формулы превращаются в код (LaTex), а таблицы и заголовки теряют структуру (md). Мне нужен был способ легко копировать из LLM в ворд с сохранением формул (хотя бы так, чтобы они читались и были понятны), поэтому был создан этот скрипт.

# llm2rtf
Скрипт берет текст из буфера обмена, вычищает мусор LaTeX, конвертирует сложные формулы в символы Unicode и упаковывает всё в Rich Text Format (RTF), который поддерживается практически всеми редакторами текста.

**Исходный текст от LLM:**
> В $\triangle ABC$ известно: $\angle \alpha = 60^\circ$, сторона $AB = \mathbf{10\sqrt{3}}\text{ см}$. Найти высоту $h = \frac{AB \cdot \sin(\alpha)}{\sqrt{x^2 + y^2}}$.

Результат при копировании из LLM в редактор **без конвертации**:
> `В $\triangle ABC$ известно: $\angle \alpha = 60^\circ$, сторона $AB = \mathbf{10\sqrt{3}}\text{ см}$. Найти высоту $h = \frac{AB \cdot \sin(\alpha)}{\sqrt{x^2 + y^2}}$.`

Результат **после конвертации**:
> В *△ ABC* известно: *∠ α = 60°*, сторона *AB = 10√3 см*. Найти высоту *h = (AB ⋅ sin(α)) / √(x² + y²)*.

### Какие редакторы поддерживаются
RTF — это стандарт форматирования, уходящий корнями в мохнатый 1987 год. Результат работы скрипта можно вставлять в:
- Microsoft Word (любых версий, включая 97, 2003, 2007 и современные).
- WordPad
- LibreOffice / OpenOffice.
- Старые редакторы типа AbiWord.
- Текстовые процессоры на macOS (TextEdit) и Linux.

### Как это работает
1. Вы копируете текст из окна чата с нейросетью (`Copy as markdown`).
2. Запускаете скрипт `llm2rtf.py` (двойным кликом, иконкой на панели, горячей клавишей или через терминал: `python llm2rtf.py` - тут как придумаете).
3. Скрипт обрабатывает текст и обновляет буфер обмена.
4. Вы вставляете результат в свой редактор (`Ctrl+V`).

## Что скрипт умеет
- [x] Автоматическая очистка мусорных тегов LaTeX (`\left`, `\quad`, `\begin`).
- [x] Превращение дробей (`\frac{a}{b}`) и вложенных корней (`\sqrt`) в читаемый Unicode-вид (`√`).
- [x] Математические символы: заменяет команды типа \alpha, \triangle, \perp на α, △, ⊥.
- [x] Конвертация индексов и степеней в надстрочные Unicode-символы (`x^2` в `x²` и `x_{12}` в `x¹²`.).
- [x] Минимальный Markdown (Заголовки, Жирный, *Курсив*). Математические переменные автоматически становятся курсивом, три уровня заголовков.
- [x] Преобразует Markdown-таблицы в таблицы RTF (с рамками и авто-шириной)
- [x] Мультипоточный буфер обмена для Windows (одновременная отправка Plain Text и RTF).

### Ограничения
Сложная верстка (например, многоэтажные дроби) упрощается для совместимости со старыми редакторами. Например: интегралы, суммы с индексами сверху/снизу и матрицы не отрисовываются графически.

Примеры упрощений:
1. Дроби: Сложные дроби записываются через слеш.
   - Было: `\frac{a + b}{c + d}`
   - Стало: `(a + b) / (c + d)`
2. Корни:
   - Было: `\sqrt{x^2 + y^2}`
   - Стало: `√(x² + y²)`

## llm2word

Вторая версия скрипта, `llm2word.py` сделана именно для ворда и пытается использовать механизм полей `EQ`, чтобы отобразить формулы в их истинном виде (с дробями, матрицами, интегралами и прочим). При вставке они превращаются в стандартные математические объекты.

### Что умеет:
- **Честная двухэтажная математика:** Дроби `\frac{a}{b}` рендерятся вертикально, а не в строку.
- **Корни любой степени:** Поддерживает как обычный `\sqrt{x}`, так и `\sqrt[n]{x}`.
- **Пределы, суммы и интегралы:** Верхние и нижние индексы встают строго на свои места (над и под знаками `\sum`, `\prod`, `\lim`, `\int`).
- **Системы уравнений и матрицы:** Окружения `cases` (фигурная скобка слева на несколько строк) и `matrix` выравниваются автоматически.
- Минимальный маркдаун и таблицы

### Как устроен код:
- **Парсинг изнутри-наружу:** Без тяжелых парсеров и AST-деревьев. Регулярные выражения находят самые глубокие вложенные скобки, сворачивают их в маркеры `\x01...\x02` и пошагово выходят наружу.
- **Автоопределение локали:** Поля `EQ` в русской винде требуют точку с запятой `;` в качестве разделителя, а в английской — запятую `,`. Скрипт сам подставляет нужный символ.

### Ограничения:
- Парсится только базовая математика, которую выдают популярные LLM.
- Эмодзи по умолчанию вырезаются.
